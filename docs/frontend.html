Frontend & UI Design

The frontend of Portfoli-AI is designed for visual impact, using a combination of Jinja2 for server-side rendering and Vanilla CSS for a high-performance, custom-tailored aesthetic.

Design System

The UI is built on a "Glassmorphism" design language, characterized by:

  • Dark Mode First: Deeper blacks and high-contrast typography.
  • Transparency: Semi-transparent backgrounds with backdrop filters (blur).
  • Typography: 'Outfit' for striking headers and 'Inter' for high legibility in body text.

AI Representative Widget

The chat widget is a complex component that handles real-time interaction with the AI backend. It features:

  • Smooth Transitions: CSS keyframe animations for message entry.
  • Bouncing Indicator: A sophisticated typing indicator to manage user expectations.
  • Dynamic Scaling: Responsive design that stays elegant on mobile viewports.
portfolio_public.html (CSS Fragment) 
.bubble {
    animation: messageSlide 0.4s cubic-bezier(0.23, 1, 0.32, 1) forwards;
}

@keyframes messageSlide {
    from { opacity: 0; transform: translateY(10px) scale(0.95); }
    to { opacity: 1; transform: translateY(0) scale(1); }
}

Scroll Animations

We use the IntersectionObserver API to trigger entrance animations as the user scrolls through the portfolio.

JavaScript Fragment 
const observer = new IntersectionObserver((entries) => {
    entries.forEach(entry => {
        if (entry.isIntersecting) {
            entry.target.style.opacity = '1';
            entry.target.style.transform = 'translateY(0)';
        }
    });
});
